Html Basic Structure of boiler plate

<!DOCTYPE html>

This is a mandatory declaration at the very top of the document that tells the browser the page is written in HTML5.If we don't include this declaration, the browser will assume the page is written in an older version of HTML.

<html>

The html tag is the root element becauew it act as a container for other elements and tell the browser every thing inside this tag is html

<head>

A container for metadata (data about the page). This information is not visible to users but is crucial for browsers and search engines.

<body>

This contains all the visible content, such as text, images,buttons etc.

A block-level element is a type of element that structures a web page by taking up the full width available and always starting on a new line
